# Sprite Clone

Sprite Clone is a mobile Bullet Heaven game developed using Unity 2022.  
Players must survive endless waves of enemies with increasing difficulty while upgrading unique skills and achieving the highest possible score.

---

# Features

- Endless survival gameplay
- Mobile twin-stick controller
- Dash mechanic
- Skill upgrade system
- Multiple unique skills:
  - Orbiting shuriken
  - Freeze enemy
  - Bigger bullets
  - And more
- Increasing enemy difficulty over time

---

# Controls

## Mobile

- Left Joystick → Move player
- Right Joystick → Aim & Auto Shoot
- Dash Button → Dash

# How to Open the Project

1. Download or clone this repository
2. Open Unity Hub
3. Click "Open"
4. Select the project folder
5. Open the project using Unity 2022.3.x LTS

---

# How to Build APK

1. Open the project in Unity
2. Go to:

```txt
File → Build Settings
```

3. Select:

```txt
Android
```

4. Click:

```txt
Switch Platform
```

5. Make sure all required scenes are added to Build Settings
6. Click:

```txt
Build
```

or

```txt
Build And Run
```

---

# How to Test the Game

## On Mobile Device

1. Install the APK file
2. Open the game
3. Use:
   - Left joystick to move
   - Right joystick to aim and shoot
   - Dash button to evade enemies
4. Survive as long as possible and get the highest score

## In Unity Editor

You can also test the game directly in Unity using the Play button.
